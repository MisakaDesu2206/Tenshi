# Tenshi
Tenshi Facebook - Coffeescript 

edit file config.js

```
{
    "username" : "{{email}}",
    "password" : "{{password}}",
    "prefix": "{{prefix}}", //ví dụ prefix là ! => bạn muốn dùng lệnh => ping => !ping
    "token" : ""
}
```

**HOW TO RUN ?**
sau khi edit file config.js xong bạn chạy lệnh dưới
```
npm i && npm start
```
